# Student Management System


## Project Description
A web-based system to manage and track student performance easily. This project allows you to add students, edit their grades, add bonuses, and delete all records if needed. It also provides automatic statistics for students, such as:
- Total number of students
- Number of excellent students
- Number of passed students
- Number of failed students

The interface is built with **HTML** and **Tailwind CSS**, and interactive functionality is implemented using **JavaScript** with **localStorage** for storing data locally.

---

## Features
- Add a new student with ID, name, attendance, mid-term and final grades, and average tasks.
- Add bonus points to students with automatic update of the final grade.
- Delete all students with confirmation before the action.
- Display studentsâ€™ data in a dynamic table.
- Automatic statistics update after each change.
- Attractive and responsive UI using Tailwind CSS.
- Toast notifications to inform users about success or errors.

## Technologies Used
- **HTML5**
- **CSS3 / Tailwind CSS**
- **JavaScript (Vanilla)**
- **LocalStorage** for storing student data locally.
